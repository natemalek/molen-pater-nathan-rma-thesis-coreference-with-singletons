Instructions

People can use pronouns (for example they, he, she), repeat a name (for example John, the US), and other ways to point to people or things in the world. Your task is to put all expressions that refer to the same thing, person, location, etc. into one group, marked by a number. For example:

[Martha]1 is very kind. [She]1 often helps [her]1 neighbors.

Not only people and objects can be referred to but also events. In those cases, we mark the verb, for example:

Robert [won]2 his third race in a year. [It]2 makes me jealous.

User Interface

  • Once you finish annotating the first sentence, the second will appear and so on.
  • Click on any highlighted expressions to select an existing group or start a new one.
  • When a group is selected (printed in bold), click on an expression to add/remove it to/from the group.
  • To save the annotation and move on to a new group (you can always come back), use the button "Save" (hot key: Enter or Space).
  • To ignore the current grouping and revert back to the last time you press "Save", use the button "Cancel".
  • To skip one expression (only when you're not certain about its grouping), use the button "Skip".
  • Each blank represents one word. Blanks are editable. While you are editing, use Tab to move to the next blank and Shift+Tab to go to the previous one. You are encouraged to fill them to support your memory but they are not used to grade your work.
  • Notice that an expression might enclose one another -- you need to put each of them in its right group.
  • These instructions can be shown using the button "Instructions".
